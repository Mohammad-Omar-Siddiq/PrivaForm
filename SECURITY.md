# Security Policy

## Privacy Guarantee

**PrivaForm processes all documents locally on your machine.** No files, data, or metadata are transmitted to any external server.

### What This Means
- ✅ TIFF input files stay on your device
- ✅ Generated PDFs are saved locally
- ✅ No internet connection required
- ✅ No telemetry, no analytics, no tracking
- ✅ Source code is publicly auditable (MIT + Commons Clause)

### How to Verify
1. Run the `.exe` with network monitoring (e.g., Wireshark)
2. Review source code at [GitHub](https://github.com/Mohammad-Omar-Siddiq/PrivaForm)
3. Check your firewall logs — no external connections will be made
